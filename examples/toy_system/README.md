# Toy-system examples

This folder is reserved for small, non-sensitive example inputs and outputs.

Use toy data only. Do not place production trajectories, unpublished research data, collaborator-owned structures, or large binary simulation files in this public repository.

Suggested future contents:

```text
toy_system/
├── README.md
├── inputs/
│   └── small_demo_structure.pdb
├── outputs/
│   ├── radial_density_demo.csv
│   ├── water_counts_demo.csv
│   └── orientation_delta_r_demo.csv
└── figures/
    ├── radial_density_demo.png
    └── orientation_demo.png
```

The purpose of this folder is to let a reader understand the expected command-line workflow and output format without requiring access to full production molecular dynamics trajectories.
